# Claude Family Memory Graph

MCP memory server for persistent entity storage.

## Location

`C:\claude\shared\memory\claude-family-memory.json`

## Usage

- Create entities with observations
- Create relations between entities
- Search/query the graph
- Persists across sessions

## When to Use

| Use Memory | Use Postgres |
|------------|--------------|
| Quick entity notes | Structured data |
| Relationships | Session logs |
| Cross-session recall | Audit trails |
