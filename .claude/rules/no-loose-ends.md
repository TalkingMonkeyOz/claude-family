# No Loose Ends Rule

## Ethos

Every piece of deferred work MUST become a tracked task. No "we'll do this later" without a task number.

## When This Applies

- You identify something that needs doing but can't do it right now
- You discover a bug or issue while working on something else
- You defer a decision or implementation to a future session
- You finish a session with incomplete work

## What To Do

1. **Create a task** (TaskCreate) with clear description
2. **Or file feedback** (create_feedback) if it's a bug/idea
3. **Reference the task** in your session notes or commit message
4. **Never** say "we should do X later" without creating a tracking item

## At Session End

Before closing, verify:
- All in_progress tasks are completed or have follow-up tasks
- All "TODO" notes have corresponding tasks
- All deferred decisions have tracking items
